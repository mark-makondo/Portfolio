## 🚀 Overview

This project serves as my digital home and professional showroom. It features a custom-themed **DaisyUI** interface, **GSAP-powered** page transitions, and a fully responsive layout optimized.

### Key Features
* **Fluid Typography:** Dynamic scaling using CSS `clamp()` for perfect readability on every device.
* **GSAP Animations:** Staggered reveals and smooth route transitions.
* **Custom Theming:** A tailored DaisyUI system with custom base-color overrides.
* **Type-Safe UI:** Custom React components built with strict TypeScript for Inputs and Textareas.

## 📦 Getting Started

### Prerequisites
* Node.js (v18 or higher)
* npm / yarn / pnpm

### Installation

1. **Clone the repo**
   ```bash
   git clone [https://github.com/](https://github.com/)[your-username]/[your-repo-name].git
   ```

2. **Install dependencies**
    ```bash
        npm install
    ```
3. Start the development server
    ```bash
        npm run dev
    ```
